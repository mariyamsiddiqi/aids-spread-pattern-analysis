# 🏥 AIDS Spread Pattern Analysis and Early Detection System Using Machine Learning

An intelligent Machine Learning-based system designed to analyze HIV/AIDS spread patterns, identify high-risk regions, and support early detection through predictive analytics and an interactive Streamlit dashboard.

---

## 📌 Project Overview

This project leverages Machine Learning and Data Science techniques to analyze HIV/AIDS trends using a global health dataset with a focus on Pakistan. It includes data preprocessing, feature engineering, predictive modeling, and an interactive dashboard that helps visualize disease patterns and supports data-driven decision-making.

---

## ✨ Key Features

- 📊 Data Cleaning and Preprocessing
- 📈 Exploratory Data Analysis (EDA)
- ⚙️ Feature Engineering
- 🤖 Machine Learning Model Development
- 🎯 HIV Spread Risk Prediction
- 🇵🇰 Pakistan-Specific Analysis
- 🌍 Global HIV/AIDS Trends Visualization
- 🖥️ Interactive Streamlit Dashboard

---

## 🤖 Machine Learning Models

The following models were trained and evaluated:

- 🌳 Random Forest
- 🚀 XGBoost *(Best Performing Model)*
- 🧠 Neural Network

| Model | Accuracy |

```bash

```

3. Run the Streamlit application.

```bash
streamlit run app.py
```
